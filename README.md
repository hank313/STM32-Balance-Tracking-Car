\# STM32 Balance Tracking Car



\## 專案簡介



本專案為 STM32 自平衡循跡車控制程式，使用 STM32CubeMX 建立專案初始化設定，並使用 Keil uVision 撰寫與編譯主要控制程式。



系統主要功能包含：



\- 使用 MPU6050 讀取車體角度與角速度

\- 使用互補濾波器估測目前車體傾斜角

\- 使用 PID 控制器計算馬達 PWM 輸出

\- 使用 TB6612FNG 馬達驅動模組控制左右馬達

\- 使用左右循跡感測器修正車體行進方向



\## 開發環境



\- STM32CubeMX

\- Keil uVision

\- STM32 HAL Library

\- C Language



\## 使用硬體



\- STM32 開發板

\- MPU6050 六軸感測器

\- TB6612FNG 馬達驅動模組

\- DC 馬達 x 2

\- 循跡感測器 x 2

\- 電池供電模組

Core/Src/main.c



\## 程式功能說明



\### 1. MPU6050 感測器讀取



程式透過 I2C 讀取 MPU6050 的加速度與陀螺儀資料，用來判斷車體目前的傾斜角度。



\### 2. 互補濾波器



使用加速度計計算角度，並結合陀螺儀角速度，透過互補濾波器取得較穩定的角度估測值。



\### 3. PID 平衡控制



程式根據目前角度與目標角度之間的誤差，計算馬達所需的 PWM 輸出，使車體能夠維持平衡。



主要 PID 參數：



Kp = 35.0  

Ki = 0.0  

Kd = 2.2  



\### 4. 循跡控制



程式讀取左右循跡感測器狀態，判斷車體是否偏離路線，並透過調整左右馬達 PWM 讓車體回到路線上。



主要循跡參數：



Forward\_Speed = 200  

K\_Track = 200.0  



\### 5. 馬達 PWM 控制



程式使用 TIM2 與 TIM3 輸出 PWM，分別控制左右馬達轉速，並使用 GPIO 控制馬達正反轉方向。



\## 如何使用



1\. 使用 STM32CubeMX 開啟 Motor\_test.ioc

2\. 檢查 GPIO、I2C、TIM PWM 腳位設定

3\. 使用 Keil uVision 開啟 Motor\_test.uvprojx

4\. 編譯專案

5\. 將程式燒錄到 STM32 開發板

6\. 接上 MPU6050、馬達驅動模組與循跡感測器

7\. 啟動系統並測試自平衡與循跡功能



\## 注意事項



\- 若車體傾斜角度超過 ±45 度，程式會停止馬達輸出，避免失控。

\- PID 參數需要依照實際車體重心、馬達轉速與電池電壓進行調整。

\- 若左右馬達速度不同，可調整馬達補償比例。

\- 測試時建議先架高車體，確認馬達方向正確後再落地測試。
